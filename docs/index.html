Databases for text analysis

doctable is a python package for parsing, storing, and accessing text documents in databases.

pip install doctable
pip install --upgrade git+https://github.com/devincornell/doctable.git@master

DocTable Objects Overview » API »

Object-oriented interface for querying and manipulating a single database table.

Examples: insert and delete / select / update / schema guide

Related functionality
  • dataclass schemas: create doctable schemas using dataclasses. (examples / api / column definitions / column type map / schema guide)
  • ConnectEngine: connect to database independent of any single table. (examples / api)
  • Bootstrap: bootstrap database rows for model training. (examples / api)
  • database functions (api)

Parsing Pipelines Overview »

Various tools for parsing and storing documents into databases.

Related functionality
  • ParsePipeline: manages parallelized parsing pipelines using custom functions or pre-built doctable components. (overview / components / api)
  • ParseTree: minimal container for storing and manipulating parsetree objects. (api)
