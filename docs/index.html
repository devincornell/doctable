Databases for text analysis

doctable is a python package for parsing, storing, and accessing text documents and models for large scale analysis.

pip install doctable

Every text analysis project is different, but they almost always share the first few steps: preprocessing, parsing/tokenization, and storage into a format that can be used for analysis. doctable is designed to facilitate these steps by providing a series of small decisions which are common to most applications. The package consists primarily of two Python classes: DocTable and DocParser. We've created some demonstrations so you can see how we use it:

US National Security Strategy Documents.

This demonstration shows a typical workflow using DocTable and DocParser. We use DocParser to download, tokenize, and create parsetrees for 17 national security strategy documents. We take advantage of DocParser to create a parallelized workflow for parsing, and a custom DocTable class for efficient storage.
